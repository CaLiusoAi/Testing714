I. FOUNDATIONAL DEFINITIONS
═══════════════════════════════════════════════════════════════════════════════

1.1 DISCRETE DYNAMICAL SYSTEM
────────────────────────────────

DEFINITION: A discrete dynamical system is a tuple (S, T) where:
  • S is a finite set of states
  • T: S → S is a deterministic transition function
  • Evolution: S(t+1) = T(S(t))

NOTATION:
  • S(t): State at timestep t
  • T^n(S): n-fold application of T to state S
  • |S|: Cardinality of state space

CONSTRAINT: This framework considers only:
  • Finite state spaces: |S| < ∞
  • Deterministic transitions: T is a function, not a relation
  • Discrete time: t ∈ ℕ

1.2 ATTRACTOR STRUCTURE
────────────────────────

DEFINITION: A state A ∈ S is an attractor if:
  • T(A) = A (fixed point), OR
  • ∃k > 0: T^k(A) = A (periodic orbit)

DEFINITION: An attractor A is GLOBAL if:
  • ∀S₀ ∈ S: ∃t: T^t(S₀) = A (all states eventually reach A)

DEFINITION: An attractor is UNIQUE if:
  • No other attractors exist in S
  • All trajectories merge to single destination

1.3 TRAJECTORY DIVERGENCE
──────────────────────────

DEFINITION: Given two trajectories S_E(t), S_L(t) starting from perturbed
initial conditions, the divergence at time t is:

  d(t) = distance(S_E(t), S_L(t))

Common distance metrics:
  • For sets: d(S₁, S₂) = |S₁ Δ S₂| (symmetric difference cardinality)
  • For integers: d(x, y) = |x - y| (absolute difference)
  • For vectors: d(x, y) = ||x - y|| (Euclidean norm)

DEFINITION: SUSTAINED DIVERGENCE occurs if:
  ∃ε > 0: lim sup_{t→∞} d(t) ≥ ε
  (Gap persists indefinitely)

DEFINITION: TRANSIENT AMPLIFICATION occurs if:
  ∃t₀, k > 0: d(t₀ + k) > d(t₀)
  (Gap grows temporarily before eventual convergence)

1.4 SPECTRAL RADIUS
────────────────────

DEFINITION: The spectral radius ρ(Δ_T) measures divergence growth rate:

  ρ(Δ_T) = lim sup_{t→∞} [d(t)]^(1/t)

INTERPRETATION:
  • ρ < 1: Exponential contraction (divergence shrinks)
  • ρ = 1: Marginal stability (divergence bounded)
  • ρ > 1: Exponential expansion (divergence grows)

CONNECTION TO AMPLIFICATION:
  • Sustained divergence ⟹ ρ ≥ 1
  • Transient amplification may occur even if ρ ≤ 1
  • ρ = 0 indicates immediate synchronization

II. THE THREE ORTHOGONAL AXES
═══════════════════════════════════════════════════════════════════════════════

This framework establishes that convergence behavior decomposes into three
independent structural properties:

2.1 AXIS 1: TOPOLOGY (ATTRACTOR STRUCTURE)
───────────────────────────────────────────

PROPERTY: Number and accessibility of attractors

QUESTION ANSWERED: "Where do trajectories end?"

CLASSIFICATION:
  • Single global attractor: All trajectories converge to same destination
  • Multiple attractors: Basin-dependent convergence
  • No attractors: Infinite wandering or unbounded growth

GUARANTEES (Single Global Attractor):
  ✓ Eventual convergence: lim_{t→∞} d(t) = 0
  ✓ No sustained divergence possible

DOES NOT GUARANTEE:
  ✗ How trajectories approach attractor
  ✗ Whether divergence increases transiently
  ✗ Rate of convergence

INDEPENDENCE: Topology is independent of descent and geometry.
  Example: Same attractor structure admits both amplifying and
  non-amplifying dynamics depending on transition function.

2.2 AXIS 2: DESCENT (LYAPUNOV STRUCTURE)
─────────────────────────────────────────

PROPERTY: Individual trajectory progress toward attractor

QUESTION ANSWERED: "Do trajectories move monotonically toward the attractor?"

DEFINITION: A strict Lyapunov function is Φ: S → ℕ such that:
  1. Φ(A) = 0 (minimal at attractor)
  2. Φ(T(x)) < Φ(x) for all x ≠ A (strict descent)

GUARANTEES (Strict Lyapunov Exists):
  ✓ No cycles (acyclic dynamics)
  ✓ Finite convergence time: ∀S₀: T^(Φ(S₀))(S₀) = A
  ✓ Well-founded descent

DOES NOT GUARANTEE:
  ✗ Non-amplification of divergence between trajectories
  ✗ Order preservation
  ✗ Geometric contraction

CRITICAL INSIGHT: Lyapunov controls VERTICAL descent (progress toward attractor)
but not HORIZONTAL divergence (separation between trajectories).

INDEPENDENCE: Two trajectories can both descend strictly in Φ while their
mutual distance increases.

2.3 AXIS 3: GEOMETRY (ORDER STRUCTURE)
───────────────────────────────────────

PROPERTY: Preservation of relationships between states

QUESTION ANSWERED: "How do trajectories relate to each other during evolution?"

DEFINITION: T is ORDER-PRESERVING if ∃ partial order ≤ on S such that:
  x ≤ y ⟹ T(x) ≤ T(y)

GUARANTEES (Order-Preserving):
  ✓ No transient amplification
  ✓ Geometric contraction: If S₁ ⊆ S₂ then d(t+1) ≤ d(t)
  ✓ No path crossovers

DOES NOT GUARANTEE:
  ✗ Existence of attractor
  ✗ Acyclic dynamics
  ✗ Specific convergence rate

CRITICAL INSIGHT: Order preservation is the ONLY property that prevents
transient amplification.

INDEPENDENCE: Order structure is independent of attractor topology and
Lyapunov descent. Systems can be order-preserving without having attractors,
and can have attractors + Lyapunov without being order-preserving.

2.4 ORTHOGONALITY THEOREM
──────────────────────────

THEOREM: Topology, Descent, and Geometry are structurally independent.

PROOF BY CONSTRUCTION:

Case 1: Topology WITHOUT Descent
  System: {0→1, 1→2, 2→0} (3-cycle)
  • Attractor: 3-cycle (topology defined)
  • Lyapunov: NONE (no strict descent, cycles exist)
  • Order: NONE (cyclic structure unorderable)

Case 2: Descent WITHOUT Geometry
  System: {0→2, 1→3, 2→1, 3→3}
  • Attractor: {3} (single global attractor)
  • Lyapunov: Φ = {0:3, 2:2, 1:1, 3:0} (strict descent)
  • Order: NONE (crossover: 0→2→1 vs 1→3)
  • Result: Transient amplification occurs

Case 3: Geometry WITHOUT Topology
  System: {0→1, 1→2, 2→3, ...} (infinite chain)
  • Attractor: NONE (infinite state space, no fixed point)
  • Lyapunov: Φ(n) = n (strict descent if modified for finite)
  • Order: x ≤ y ⟺ x ≤ y in ℕ (order-preserving under ≤)

Case 4: Topology + Descent WITHOUT Geometry
  System: Integer counterexample above (Section 2.2)
  • Has attractor + Lyapunov but NOT order-preserving
  • Demonstrates independence

Case 5: Topology + Geometry WITHOUT Descent (Lyapunov)
  System: {0→1, 1→0, 2→2}
  • Attractor: {0,1} 2-cycle and {2} fixed point
  • Order: x ≤ y ⟺ x ≤ y in {0,1,2}
  • Lyapunov: NONE for 2-cycle component

Case 6: Descent + Geometry WITHOUT Global Attractor
  System: Multiple basins with order-preserving dynamics
  • Lyapunov: Local descent to multiple attractors
  • Order: Preserved in each basin
  • Global attractor: NONE

QED: All eight combinations of (Topology, Descent, Geometry) presence/absence
are realizable, proving structural independence.

3.3 KEY THEOREM: THE AMPLIFICATION BARRIER
───────────────────────────────────────────

THEOREM: Order Preservation is Necessary and Sufficient for Preventing
         Transient Amplification in Finite Deterministic Systems

STATEMENT:
  Let (S, T) be a finite deterministic system with single global attractor.
  
  SUFFICIENT DIRECTION (PROVEN):
    If T is order-preserving, then ∀t: d(t+1) ≤ d(t)
    (No transient amplification)
  
  NECESSITY (CONJECTURED):
    If ∀t: d(t+1) ≤ d(t), then T admits an order-preserving embedding
    (Open problem - see Section VII)

PROOF (SUFFICIENT DIRECTION):

  Assume: ∃ partial order ≤ on S such that x ≤ y ⟹ T(x) ≤ T(y)
  
  Given: Two trajectories S_E(t), S_L(t) with divergence d(t) = |S_E(t) Δ S_L(t)|
  
  Step 1: Partition symmetric difference
    S_E(t) Δ S_L(t) = (S_E(t) \ S_L(t)) ∪ (S_L(t) \ S_E(t))
  
  Step 2: Apply order preservation
    Let x ∈ S_E(t) \ S_L(t). If S_E(t) ≤ S_L(t), then:
      • Either T(x) ∈ T(S_E(t)) ≤ T(S_L(t)) → x absorbed
      • Or T(x) ∈ T(S_E(t)) \ T(S_L(t)) → remains in difference
    
    Crucially: No NEW elements enter the symmetric difference
    because order-preservation prevents crossovers
  
  Step 3: Monotone decrease
    d(t+1) = |T(S_E(t)) Δ T(S_L(t))|
           ≤ |S_E(t) Δ S_L(t)|
           = d(t)
    
    Equality only when symmetric difference stable
    Strict inequality when elements merge
  
  Step 4: No amplification
    Since d(t+1) ≤ d(t) for all t:
      • No transient growth: d(t+k) ≤ d(t) ∀k ≥ 0
      • Monotone convergence to 0 (by single attractor)

QED

COUNTEREXAMPLE (Showing Lyapunov Insufficient):

  System: {0→2, 1→3, 2→1, 3→3}
  
  Step 1: Verify single global attractor
    • 3 is fixed point: T(3) = 3
    • All states reach 3:
        0 → 2 → 1 → 3
        1 → 3
        2 → 1 → 3
    ✓ Single global attractor
  
  Step 2: Verify strict Lyapunov
    • Define Φ: {0→3, 2→2, 1→1, 3→0}
    • Check descent:
        Φ(0) = 3 > Φ(T(0)) = Φ(2) = 2 ✓
        Φ(1) = 1 > Φ(T(1)) = Φ(3) = 0 ✓
        Φ(2) = 2 > Φ(T(2)) = Φ(1) = 1 ✓
        Φ(3) = 0 = Φ(T(3)) = Φ(3) = 0 ✓ (attractor)
    ✓ Strict Lyapunov exists
  
  Step 3: Check for order-preservation
    • Attempt to construct partial order ≤:
        Need: 0 ≤ 2 (since 0→2)
        Need: 2 ≤ 1 (since 2→1)
        Need: 1 ≤ 3 (since 1→3)
        
        Also need: 1 ≤ 3 (given)
        But have: 2 ≤ 1 and 0 ≤ 2
        
        If order-preserving: 0 ≤ 2 ≤ 1 ≤ 3
        But T(0)=2 and T(1)=3 and T(2)=1
        
        CONTRADICTION: 2 ≤ 1 but T(0)=2 and T(2)=1
        This creates crossover: path from 0 crosses path from 1
    
    ✗ NO order-preserving embedding exists
  
  Step 4: Verify transient amplification
    • Trajectories:
        A: 0 → 2 → 1 → 3
        B: 1 → 3 → 3 → 3
    
    • Distances:
        d(0) = |0 - 1| = 1
        d(1) = |2 - 3| = 1
        d(2) = |1 - 3| = 2  ← AMPLIFICATION
        d(3) = |3 - 3| = 0  ← convergence
    
    ✓ Transient amplification occurs at t=2

  CONCLUSION:
    System has:
      ✓ Single global attractor (Level 0)
      ✓ Strict Lyapunov (Level 1)
      ✗ NOT order-preserving (fails Level 2)
      ✓ Exhibits transient amplification
    
    PROVES: Lyapunov descent insufficient to prevent amplification
            Order preservation is necessary

QED (Counterexample Complete)

4.1 EXPERIMENTAL FRAMEWORK
──────────────────────────

OBJECTIVE: Systematically test boundaries between hierarchy levels

METHOD: Deterministic state-transition engine with:
  • Explicit state space definition
  • Fixed transition operator T
  • Perturbation protocol (vary timing)
  • Divergence tracking d(t)

PROTOCOL:
  1. Define state space S and predicate set P
  2. Specify transition rules R₁, R₂, R₃, ...
  3. Set initial condition S(0)
  4. Execute two trajectories:
     - Early perturbation at time τ_E
     - Late perturbation at time τ_L
  5. Measure divergence: d(t) = |S_E(t) Δ S_L(t)|
  6. Compute cumulative divergence: D = Σ d(t)
  7. Verify hierarchy level properties

4.2 EXPERIMENT 1: PURE ABSORPTION (LEVEL 4)
────────────────────────────────────────────

SPECIFICATION:

  State Space: P = {A, B, C}
  Initial: S(0) = {A, B, C}
  
  Rules (applied in order):
    R1: IF {A, B} ⊆ S THEN S := S \ {B}
    R2: IF {B, C} ⊆ S THEN S := S \ {C}
  
  Properties:
    • Monotone: T(S) ⊆ S (only removals)
    • Idempotent: T(T(S)) = T(S) (fixed points immediate)
  
  Perturbations:
    τ_E = 0: Remove C at start
    τ_L = 5: Remove C at t=5
  
  Horizon: H = 10

EXECUTION:

  Trajectory Early (τ_E = 0):
    t=0: {A,B,C} → Remove C → {A,B}
         Apply R1: {A,B}⊆S → Remove B → {A}
         Apply R2: Not applicable
         Result: S_E(0) = {A}
    
    t=1..10: S_E(t) = {A} [FIXED POINT]
  
  Trajectory Late (τ_L = 5):
    t=0: {A,B,C}
         Apply R1: Remove B → {A,C}
         Apply R2: Not applicable
         Result: S_L(0) = {A,C}
    
    t=1..4: S_L(t) = {A,C} [STABLE]
    
    t=5: {A,C} → Remove C → {A}
         Apply R1: Not applicable
         Apply R2: Not applicable
         Result: S_L(5) = {A}
    
    t=6..10: S_L(t) = {A} [FIXED POINT]

DIVERGENCE ANALYSIS:

  t  | S_E(t) | S_L(t) | d(t) = |S_E Δ S_L|
  ---|--------|--------|-------------------
  0  | {A}    | {A,C}  | 1 (C)
  1  | {A}    | {A,C}  | 1 (C)
  2  | {A}    | {A,C}  | 1 (C)
  3  | {A}    | {A,C}  | 1 (C)
  4  | {A}    | {A,C}  | 1 (C)
  5  | {A}    | {A}    | 0
  6  | {A}    | {A}    | 0
  ...
  10 | {A}    | {A}    | 0
  
  Cumulative divergence: D = 5
  Maximum divergence: max d(t) = 1
  Convergence time: t_conv = 5 (equals perturbation delay)

PROPERTIES VERIFIED:

  ✓ Single global attractor: {A}
  ✓ Monotone decrease: d(t+1) ≤ d(t) ∀t
  ✓ No transient amplification: max d(t) = 1 (never grows)
  ✓ Bounded convergence: t_conv = O(|P|)
  ✓ Spectral radius: ρ = 1 (marginal, no growth)

CONCLUSION: Level 4 (Absorbing) verified experimentally.

5.1 THEOREM: MONOTONE CONTRACTION IN FINITE SYSTEMS
────────────────────────────────────────────────────

THEOREM:
  Let (S, T) be a finite deterministic system where:
    1. S finite
    2. T deterministic
    3. T monotone: S₁ ⊆ S₂ ⟹ T(S₁) ⊆ T(S₂)
    4. Unique global attractor A
  
  Then for any two trajectories S_E(t), S_L(t) with symmetric difference
  d(t) = |S_E(t) Δ S_L(t)|:
  
    d(t+1) ≤ d(t) for all t

PROOF:

  Let S_E(t), S_L(t) be two states at time t.
  
  Step 1: Partition symmetric difference
    S_E(t) Δ S_L(t) = [S_E(t) \ S_L(t)] ∪ [S_L(t) \ S_E(t)]
    
    Define:
      E_only(t) := S_E(t) \ S_L(t)  (elements unique to Early)
      L_only(t) := S_L(t) \ S_E(t)  (elements unique to Late)
    
    Then: d(t) = |E_only(t)| + |L_only(t)|
  
  Step 2: Apply monotone transition
    Consider element e ∈ E_only(t).
    
    Case 2a: e is removed by T from S_E(t)
      Then T(e) ∉ T(S_E(t))
      Contribution to d(t+1): 0 (element eliminated)
    
    Case 2b: e persists in T(S_E(t))
      Then T(e) ∈ T(S_E(t))
      
      Subcase 2b-i: T(e) ∈ T(S_L(t))
        Then e absorbed into common part
        Contribution to d(t+1): 0
      
      Subcase 2b-ii: T(e) ∉ T(S_L(t))
        Then T(e) ∈ E_only(t+1)
        Contribution to d(t+1): 1
    
    Key observation: No NEW elements can appear in E_only(t+1)
    that were not derivable from E_only(t).
    
    Monotonicity ensures:
      If x ∉ S_E(t) \ S_L(t) at time t,
      then T(x) cannot appear in T(S_E(t)) \ T(S_L(t)) at time t+1
      without also appearing in T(S_L(t)).
  
  Step 3: Count elements in next symmetric difference
    E_only(t+1) ⊆ T(E_only(t))  (monotone mapping)
    L_only(t+1) ⊆ T(L_only(t))
    
    Therefore:
      |E_only(t+1)| ≤ |E_only(t)|
      |L_only(t+1)| ≤ |L_only(t)|
    
    Hence:
      d(t+1) = |E_only(t+1)| + |L_only(t+1)|
             ≤ |E_only(t)| + |L_only(t)|
             = d(t)

QED

COROLLARY: In monotone systems with single attractor, ρ ≤ 1.

5.2 THEOREM: LYAPUNOV DOES NOT IMPLY GEOMETRIC CONTRACTION
───────────────────────────────────────────────────────────

THEOREM:
  ∃ finite deterministic system (S, T) such that:
    1. Has single global attractor
    2. Has strict Lyapunov function Φ
    3. Exhibits transient amplification

PROOF BY CONSTRUCTION:

  Define system:
    S = {0, 1, 2, 3}
    T = {0→2, 1→3, 2→1, 3→3}
  
  Part 1: Verify single global attractor
    T(0) = 2, T(2) = 1, T(1) = 3, T(3) = 3
    
    Trajectories:
      0 → 2 → 1 → 3 → 3 → ...
      1 → 3 → 3 → ...
      2 → 1 → 3 → 3 → ...
      3 → 3 → ...
    
    All trajectories reach 3 and remain.
    ✓ Unique global attractor: A = {3}
  
  Part 2: Construct strict Lyapunov function
    Define Φ: S → ℕ:
      Φ(0) = 3
      Φ(2) = 2
      Φ(1) = 1
      Φ(3) = 0
    
    Verify strict descent:
      Φ(T(0)) = Φ(2) = 2 < 3 = Φ(0) ✓
      Φ(T(2)) = Φ(1) = 1 < 2 = Φ(2) ✓
      Φ(T(1)) = Φ(3) = 0 < 1 = Φ(1) ✓
      Φ(T(3)) = Φ(3) = 0 = 0 = Φ(3) ✓ (attractor)
    
    ✓ Strict Lyapunov function exists
  
  Part 3: Demonstrate transient amplification
    Consider trajectories starting at 0 and 1:
    
    Trajectory A: 0 → 2 → 1 → 3
    Trajectory B: 1 → 3 → 3 → 3
    
    Distance metric: d(x,y) = |x - y|
    
    Compute divergence:
      t=0: d(0,1) = |0 - 1| = 1
      t=1: d(2,3) = |2 - 3| = 1
      t=2: d(1,3) = |1 - 3| = 2  ← AMPLIFICATION
      t=3: d(3,3) = |3 - 3| = 0
    
    Observation: d(2) = 2 > 1 = d(1)
    ✓ Transient amplification occurs
  
  Part 4: Verify Lyapunov descent during amplification
    At t=1:
      Φ(A(1)) = Φ(2) = 2
      Φ(B(1)) = Φ(3) = 0
      Both trajectories descending individually
    
    At t=2:
      Φ(A(2)) = Φ(1) = 1 < 2  (A descended)
      Φ(B(2)) = Φ(3) = 0 = 0  (B at attractor)
      
    CRITICAL: Both Lyapunov values decreased (or stable at attractor),
              yet distance d increased from 1 to 2.
    
    This proves vertical descent (Lyapunov) is independent from
    horizontal divergence (pairwise distance).

QED

COROLLARY: Strict Lyapunov function is insufficient to prevent
           transient amplification.

5.3 THEOREM: ATTRACTOR CONVERGENCE
───────────────────────────────────

THEOREM:
  In any finite deterministic system with single global attractor A:
    lim_{t→∞} d(t) = 0
  for any two trajectories S_E(t), S_L(t).

PROOF:

  Assume: (S, T) finite, deterministic, single global attractor A.
  
  Step 1: Both trajectories reach attractor
    By definition of global attractor:
      ∃t_E: ∀t ≥ t_E: S_E(t) = A
      ∃t_L: ∀t ≥ t_L: S_L(t) = A
    
    (Finiteness ensures reachability in finite time)
  
  Step 2: After both at attractor, divergence vanishes
    Let t_max = max(t_E, t_L)
    
    For all t ≥ t_max:
      S_E(t) = A
      S_L(t) = A
      
      Therefore: d(t) = |S_E(t) Δ S_L(t)| = |A Δ A| = 0
  
  Step 3: Limit is zero
    lim_{t→∞} d(t) = lim_{t→∞} 0 = 0

QED

IMPORTANT: This theorem guarantees eventual convergence (topology)
           but says NOTHING about transient behavior (geometry).

5.4 THEOREM: ORDER-PRESERVATION IS SUFFICIENT
──────────────────────────────────────────────

THEOREM:
  If T preserves a partial order ≤, then no transient amplification occurs.

PROOF:

  Assume: ∃ partial order ≤ on S such that:
           x ≤ y ⟹ T(x) ≤ T(y)
  
  Given: Two trajectories S_E(t), S_L(t)
  
  Step 1: Establish order relationship at t=0
    Either S_E(0) ≤ S_L(0), or S_L(0) ≤ S_E(0), or incomparable.
    
    WLOG assume S_E(0) ≤ S_L(0) (other cases symmetric).
  
  Step 2: Order preserved at all future times
    By induction:
      Base: S_E(0) ≤ S_L(0) (assumption)
      
      Step: Assume S_E(t) ≤ S_L(t)
            Then T(S_E(t)) ≤ T(S_L(t)) by order-preservation
            Therefore S_E(t+1) ≤ S_L(t+1)
    
    Conclusion: S_E(t) ≤ S_L(t) for all t
  
  Step 3: Symmetric difference non-increasing
    Since S_E(t) ≤ S_L(t):
      S_E(t) Δ S_L(t) = (S_L(t) \ S_E(t)) ∪ (S_E(t) \ S_L(t))
                       = (S_L(t) \ S_E(t))  (second term empty)
    
    At t+1:
      S_E(t+1) Δ S_L(t+1) = S_L(t+1) \ S_E(t+1)
                          = T(S_L(t)) \ T(S_E(t))
    
    By monotonicity of T:
      T(S_E(t)) ⊆ T(S_L(t))
      
      Therefore:
        |T(S_L(t)) \ T(S_E(t))| ≤ |S_L(t) \ S_E(t)|
    
    Hence: d(t+1) ≤ d(t)
  
  Step 4: No amplification
    Since d(t+1) ≤ d(t) for all t:
      No transient growth possible
      Divergence monotonically decreases (or stable)

QED

5.5 THEOREM: FINITE DESCENDING CHAINS IN MONOTONE SYSTEMS
──────────────────────────────────────────────────────────

THEOREM:
  In finite monotone systems, all trajectories reach fixed points
  in at most |S| steps.

PROOF:

  Assume: S finite, T monotone: T(S) ⊆ S
  
  Step 1: Strict subset descent
    If S ≠ T(S), then T(S) ⊂ S (proper subset)
  
  Step 2: Cardinality decreases
    If T(S) ⊂ S, then |T(S)| < |S|
  
  Step 3: Bounded descent
    Starting from S₀:
      |S₀| > |T(S₀)| > |T²(S₀)| > ... > |T^k(S₀)|
    
    This is a strictly decreasing sequence of natural numbers.
    
    Since |S₀| ≤ |2^P| = 2^|P|, and each step decreases by at least 1:
      Maximum descent length = |S₀| ≤ |P|
  
  Step 4: Fixed point reached
    After at most |P| steps, must reach T^k(S₀) = T^(k+1)(S₀)
    This is a fixed point.

QED

COROLLARY: In absorbing systems (Level 4), convergence time is O(|P|).

6.1 LEVEL 1 WITHOUT LEVEL 2: LYAPUNOV BUT NOT ORDER-PRESERVING
───────────────────────────────────────────────────────────────

SYSTEM: Integer state space with crossover paths

  State Space: S = {0, 1, 2, 3}
  Transition: T(0)=2, T(1)=3, T(2)=1, T(3)=3
  Distance: d(x,y) = |x - y|

VERIFICATION:

  Single Global Attractor:
    All states reach 3:
      0 → 2 → 1 → 3 → 3 → ...
      1 → 3 → 3 → ...
      2 → 1 → 3 → 3 → ...
      3 → 3 → ...
    ✓ Level 0 satisfied

  Strict Lyapunov Function:
    Φ(0)=3, Φ(2)=2, Φ(1)=1, Φ(3)=0
    
    Descent verification:
      Φ(T(0)) = Φ(2) = 2 < 3 = Φ(0) ✓
      Φ(T(2)) = Φ(1) = 1 < 2 = Φ(2) ✓
      Φ(T(1)) = Φ(3) = 0 < 1 = Φ(1) ✓
      Φ(T(3)) = Φ(3) = 0 = Φ(3) ✓
    ✓ Level 1 satisfied

  Order-Preservation Test:
    Attempt to construct ≤:
      Need: 0 ≤ 2 (since 0→2)
      Need: 2 ≤ 1 (since 2→1)
      Need: 1 ≤ 3 (since 1→3)
      Need: 0 ≤ 1 (both start near each other)
      
      From 0 ≤ 2 and 2 ≤ 1: get 0 ≤ 1 (transitive)
      From 0 ≤ 1 and 1 ≤ 3: get 0 ≤ 3 ✓
      
      But 2 ≤ 1 conflicts with standard order on integers!
      
      If we try natural order: 0 < 1 < 2 < 3
        Then T(0)=2 and T(1)=3 satisfies 2 < 3 ✓
        But T(1)=3 and T(2)=1 gives 3 > 1
        Violates order-preservation: 1 < 2 but T(1)=3 > 1=T(2)
      
      No consistent partial order exists.
    ✗ Level 2 NOT satisfied

  Transient Amplification:
    Trajectories from 0 and 1:
      t=0: d(0,1) = 1
      t=1: d(2,3) = 1
      t=2: d(1,3) = 2  ← AMPLIFICATION
      t=3: d(3,3) = 0
    ✓ Amplification occurs (as predicted by failing Level 2)

CONCLUSION: This system is Level 1 but NOT Level 2.
            Proves Lyapunov insufficient for preventing amplification.

6.2 LEVEL 0 WITHOUT LEVEL 1: ATTRACTOR BUT CYCLES
──────────────────────────────────────────────────

SYSTEM: Cyclic dynamics with eventual convergence

  State Space: S = {A, B, C, D}
  Transition: T(A)=B, T(B)=A, T(C)=D, T(D)=D

VERIFICATION:

  Attractor Analysis:
    Fixed point: {D} (T(D)=D)
    Cycle: {A, B} (T(A)=B, T(B)=A)
    
    Trajectories:
      A → B → A → B → ...  (2-cycle, never reaches D)
      B → A → B → A → ...  (2-cycle)
      C → D → D → ...      (reaches attractor)
      D → D → ...          (at attractor)
    
    NO global attractor: A and B never escape cycle.
    But if we modify to T(B)=D (breaking cycle):
      Then all reach D.
  
  Modified system: T(A)=B, T(B)=D, T(C)=D, T(D)=D
    ✓ Level 0 satisfied (global attractor {D})

  Lyapunov Test:
    Original (with cycle): No strict Lyapunov possible
      Φ(A) and Φ(B) must satisfy Φ(A) < Φ(B) and Φ(B) < Φ(A)
      Contradiction.
    
    ✗ Level 1 NOT satisfied (cycle prevents Lyapunov)

CONCLUSION: Cycles prevent Lyapunov descent.
            Level 0 ≠ Level 1 in general.

6.3 MONOTONE WITHOUT SINGLE ATTRACTOR
──────────────────────────────────────

SYSTEM: Multiple absorbing fixed points

  State Space: S = 2^{A,B,C}
  Initial: varies
  
  Rules:
    R1: IF A ∈ S THEN S := S \ {B, C}
    R2: IF B ∈ S AND A ∉ S THEN S := S \ {C}

VERIFICATION:

  Monotonicity:
    Both rules only remove elements.
    T(S) ⊆ S always.
    ✓ Monotone

  Attractors:
    Starting from {A, B, C}:
      R1 fires: {A}
      Fixed point reached.
    
    Starting from {B, C}:
      R2 fires: {B}
      Fixed point reached.
    
    Two attractors: {A} and {B}
    ✗ No single global attractor

CONCLUSION: Monotonicity does not imply single attractor.
            Can have multiple basins with monotone dynamics.

6.4 ORDER-PRESERVING WITHOUT ATTRACTORS
────────────────────────────────────────

SYSTEM: Infinite ascending chain

  State Space: S = ℕ (natural numbers)
  Transition: T(n) = n + 1

VERIFICATION:

  Order Preservation:
    Natural order: x ≤ y in ℕ
    T preserves: x ≤ y ⟹ x+1 ≤ y+1 ✓
    ✓ Order-preserving

  Attractor Test:
    No fixed point: T(n) ≠ n for any n
    No cycles: Sequence 0→1→2→3→... never repeats
    ✗ No attractor exists

CONCLUSION: Order-preservation independent of attractor existence.
            (Requires finiteness for attractor guarantees)

7.1 NECESSITY OF ORDER-PRESERVATION
────────────────────────────────────

CONJECTURE:
  If (S, T) is finite, deterministic, with single global attractor, and
  d(t+1) ≤ d(t) for all t (no transient amplification), then T admits
  an order-preserving embedding.

STATUS: OPEN

DISCUSSION:
  • Sufficient direction proven (Section 5.4)
  • Converse unclear: Could geometric contraction arise without order?
  • Possible weaker structures: quasi-orders, metrics, potential functions
  • May require metric-specific analysis

APPROACH:
  1. Characterize graph properties forcing order-realizability
  2. Identify obstructions (crossing paths, incomparable states)
  3. Prove or find counterexample

IMPORTANCE: Determines whether order-preservation is THE fundamental
            barrier or merely A sufficient condition.

7.2 CHARACTERIZATION OF ORDER-REALIZABLE GRAPHS
────────────────────────────────────────────────

PROBLEM:
  Given finite directed graph G with unique sink, determine when G
  admits partial order ≤ making it order-preserving.

FORMAL STATEMENT:
  Input: Directed graph (V, E)
  Question: ∃ partial order ≤ on V such that (u,v) ∈ E ⟹ u ≤ v?

KNOWN RESULTS:
  • Acyclic graphs (DAGs) always admit topological ordering
  • Cycles prevent order-embedding
  • Crossing paths create obstructions

OPEN QUESTIONS:
  1. Minimal obstructions: What subgraph patterns prevent ordering?
  2. Efficient algorithm: Can order-realizability be checked in polynomial time?
  3. Relaxations: Quasi-orders, preorders, tolerance relations?

APPLICATIONS:
  • Dynamical systems classification
  • Reachability in state machines
  • Lattice completion problems

7.3 MINIMAL GEOMETRIC CONDITIONS FOR CONTRACTION
